# Temporary

## Doel

Deze map bevat tijdelijke documenten die slechts gedurende een korte periode nodig zijn.

De inhoud van deze map is niet bedoeld als permanente projectdocumentatie.

---

# Gebruik

Gebruik deze map voor:

- Tijdelijke notities
- Werkdocumenten
- Concepten
- Testbestanden
- Snelle lijstjes
- Tijdelijke exports

---

# Richtlijnen

Controleer deze map regelmatig.

Verwijder bestanden die niet meer nodig zijn.

Verplaats waardevolle informatie naar de juiste locatie binnen de documentatie.

---

# Niet opslaan

Gebruik deze map niet voor:

- Definitieve documentatie
- Architectuur
- Procedures
- Templates
- Changelog
- Databaseontwerpen

---

# Doelstelling

Deze map voorkomt dat tijdelijke informatie tussen de permanente projectdocumentatie terechtkomt.{\rtf1}
