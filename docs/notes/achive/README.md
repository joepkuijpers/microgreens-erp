# Archive

## Doel

Deze map bevat oudere documenten die niet langer actief gebruikt worden, maar wel bewaard moeten blijven voor historische doeleinden.

---

# Gebruik

Gebruik deze map voor:

- Oude notities
- Afgeronde onderzoeken
- Verouderde ideeën
- Oude ontwerpen
- Gearchiveerde besluiten

---

# Richtlijnen

Bestanden in deze map worden niet meer aangepast.

Nieuwe informatie wordt altijd opgeslagen buiten het archief.

---

# Archiveren

Archiveer documenten wanneer:

- Het onderwerp is afgerond.
- Het document vervangen is.
- De informatie alleen nog historisch relevant is.

---

# Bestandsnaam

Behoud indien mogelijk de oorspronkelijke bestandsnaam.

Voeg eventueel een datum toe.

Voorbeeld:

2026-05-18_Prototype_Irrigation.md

---

# Doelstelling

Deze map bewaart de geschiedenis van het project zonder de actieve documentatie onoverzichtelijk te maken.
