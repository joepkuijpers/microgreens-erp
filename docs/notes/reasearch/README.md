# Research

## Doel

Deze map bevat alle onderzoeksdocumenten die gebruikt worden tijdens de ontwikkeling van het Microgreens ERP-project.

Onderzoek vormt de basis voor technische keuzes, innovaties en toekomstige uitbreidingen.

---

# Gebruik

Gebruik deze map voor:

- Wetenschappelijke artikelen
- Experimenten
- Testresultaten
- Vergelijkingen
- Datasheets
- Whitepapers
- Literatuuronderzoek
- Nieuwe technieken
