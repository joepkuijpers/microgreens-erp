# Troubleshooting

## Doel

Deze map bevat documentatie over problemen die tijdens de ontwikkeling van het Microgreens ERP-project zijn ontdekt en opgelost.

Door oplossingen vast te leggen worden dezelfde fouten in de toekomst sneller herkend en opgelost.

- Git-problemen
- PHP-fouten
- SQL-fouten
- Raspberry Pi-problemen
- Sensorproblemen
- Hardwareproblemen
- Deploymentproblemen
- Prestatieproblemen
