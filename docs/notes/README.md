# Notes

## Doel

Deze map bevat tijdelijke notities die ontstaan tijdens de ontwikkeling van het Microgreens ERP-project.

De inhoud van deze map is bedoeld als persoonlijk werkmateriaal en hoeft niet altijd onderdeel te worden van de officiële projectdocumentatie.

---

# Gebruik

Gebruik deze map voor:

- Snelle ideeën
- Brainstormsessies
- Tijdelijke onderzoeksnotities
- TODO-lijsten
- Probleemanalyses
- Testresultaten
- Losse aantekeningen

Wanneer informatie definitief wordt, verplaats deze dan naar de juiste documentatie binnen de map `/docs`.

---

# Richtlijnen

Gebruik duidelijke bestandsnamen.

Voorbeelden:

```
2026-07-12_Labor_Module_Ideas.md
2026-07-14_SQL_Notes.md
2026-07-18_Hardware_Testing.md
```

Werk bij voorkeur met één onderwerp per document.

---

# Niet opslaan

Deze map is niet bedoeld voor:

- Definitieve documentatie
- Architectuurdocumenten
- Databaseontwerpen
- Procedures
- Changelogs
- Templates

Deze horen thuis in de daarvoor bestemde mappen binnen `/docs`.

---

# Archiveren

Controleer deze map regelmatig.

Verwijder verouderde notities of verplaats waardevolle informatie naar:

- Project Knowledge Base
- Project Operating Manual
- Architectuur
- Database Planning
- Roadmap
- Changelog

---

# Doelstelling

Deze map fungeert als het digitale kladblok van het project.

Door tijdelijke notities gescheiden te houden van de officiële documentatie blijft de kennisbank overzichtelijk, actueel en eenvoudig te onderhouden.
